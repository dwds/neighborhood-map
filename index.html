Hi! This web app uses Knockout.js (an MVVM framework), the Google Maps API, and the Instagram API to display a filterable map of locations with the most recent public Instagram post tagged at that location. Since I created this project, Instagram deprecated the part of their API that allows accessing public posts, so that part of the app no longer functions! I have left it in for demonstraion purposes. You can view the code on GitHub.
